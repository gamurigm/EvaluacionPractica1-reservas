\section{Conclusiones}

\begin{itemize}
    \item La aplicación sistemática de técnicas de pruebas permitió identificar y resolver \textbf{5 vulnerabilidades de seguridad} y \textbf{3 hallazgos de mantenibilidad} críticos, eliminando completamente los riesgos detectados por OWASP ZAP y SonarQube.
    
    \item Las pruebas de estrés con JMeter determinaron que el sistema mantiene su estabilidad hasta los \textbf{500 usuarios concurrentes}, con una degradación catastrófica al alcanzar los 1000 usuarios (47\% de error), lo que define un umbral claro para el escalado de infraestructura.
    
    \item El incremento de la cobertura de pruebas al \textbf{82\%} mediante Jest asegura que los flujos críticos de la API están protegidos contra regresiones, facilitando el mantenimiento evolutivo del sistema de reservas.
\end{itemize}
